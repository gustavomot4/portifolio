TapGo

Descrição:

O projeto TapGo é um jogo de pênaltis no qual o jogador escolhe um time, e o sistema realiza um sorteio pseudoaleatório para definir um adversário diferente do time escolhido. Esse sistema foi desenvolvido como meu projeto individual no primeiro semestre do curso de Análise e Desenvolvimento de Sistemas.
